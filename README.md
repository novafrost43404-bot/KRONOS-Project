# KRONOS-Project

An advanced utility and automation mod built for the Fabric toolchain. Designed to streamline survival tasks, inventory optimization, and world scanning.

## Features
* **AutoTotem:** Automatic totem replacement logic for emergency survival scenarios.
* **OreScanner:** Visual scanning system to detect targeted blocks and ore matrices.
* **MiningEngine & PlacementEngine:** Advanced automation systems for block interactions and world breaking/placing.
* **Inventory Management:** Smart inventory cleanup, layout organization, and automatic hotbar refueling.
* **Keybind System:** Custom runtime configurations mapped to tailored task sequences.

## Installation
1. Make sure you have the correct version of **Fabric Loader** installed.
2. Drop the compiled `.jar` file into your `.minecraft/mods` folder.
3. Launch the game.

## Developer Notice
The environment relies on Gradle for builds. If you are compiling from source or running locally, make sure your JDK path points to a valid Java 17 package.
